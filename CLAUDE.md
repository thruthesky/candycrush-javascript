# Workflow

개발자가 요청하면 반드시 이 워크플로에 따라서 작업을 해야합니다. 만약, 이 워크플로에 따라서 작업을 할 없는 요청을 하면, 작업을 거부하고 종료해주세요.

- [ ] 작업을 시작하기 전에, 반드시 candy-skill 문서를 읽어주세요.
- [ ] 작업이 끝난 다음에 반드시 `doc` agent 와 `git` agent 를 병렬로 실행하여 작업한 내용을 candy-skill 문서에 업데이트 해 주고, git 에 커밋과 푸시를 해 주세요.
